# task-Repeater
A Python-based automation tool that records mouse movements, clicks, keyboard inputs, and timing information, converts them into a structured action skeleton, and replays the workflow with the original execution pattern.
A lightweight Python automation tool that records user interactions and replays them automatically.

Features
Record mouse movements
Record mouse clicks
Record keyboard inputs
Capture action timing and delays
Generate a reusable action skeleton
Replay recorded workflows automatically
Simple and extensible architecture
How It Works
Recorder

The recorder tracks:
Mouse movement
Mouse clicks
Keyboard events
Timing between actions

#Limitations

This project is coordinate-based and may fail if:

Screen resolution changes
Window positions change
UI layouts are modified
Website/application updates alter element locations

Future improvements may include computer vision-based element detection for more robust automation.
